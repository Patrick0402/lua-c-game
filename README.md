# Meu Jogo em C com SDL e Lua

## Compilação

É necessário ter instalados:

- GCC (ou outro compilador C)
- SDL2
- Lua (versão 5.4 recomendada)
- make

Para compilar acesse a pasta do projeto e rode:

```bash
make
```

Para limpar arquivos temporários:

```bash
make clean
```

Para rodar

```bash
./game
```
